¿Qué es Git?

Git es un sistema de control de versiones que permite gestionar cambios en proyectos de software.

Instalación

Descarga Git desde su sitio oficial e instálalo en tu computadora.

Comandos Básicos

  • git init - Inicializa un repositorio
  • git add . - Agrega archivos al área de preparación
  • git commit -m "mensaje" - Guarda cambios
  • git status - Muestra el estado del repositorio

Subir a GitHub

  1. Crear repositorio en GitHub
  2. Conectar con git remote add origin URL
  3. Subir con git push -u origin main
