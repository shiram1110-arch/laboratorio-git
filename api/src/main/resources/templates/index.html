Aprende Git desde cero 🚀

¿Qué es Git?

Git es un sistema de control de versiones distribuido que permite registrar cambios en archivos y coordinar trabajo en equipo.

¿Por qué usar Git?

  • Control de versiones
  • Trabajo colaborativo
  • Historial de cambios
  • Recuperación de versiones

Comandos Básicos

git init

Inicializa repositorio

git add .

Agrega archivos

git commit

Guarda cambios

git status

Estado del repo

Pasos para subir a GitHub

  1. Crear repositorio
  2. git remote add origin URL
  3. git push -u origin main

Ejemplo de flujo

git init
git add .
git commit -m "Primer commit"
git push
